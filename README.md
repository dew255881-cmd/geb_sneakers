# 👟 Geb Sneakers - เว็บไซต์ร้านขายรองเท้าออนไลน์
โปรเจคเว็บไซต์ระบบจัดการร้านขายรองเท้า พัฒนาด้วย PHP (Native) และ MySQL 

## 🌟 ฟีเจอร์หลัก (Features)
- **ระบบสมาชิก:** สมัครสมาชิก, เข้าสู่ระบบ, จัดการโปรไฟล์ (อัปโหลดรูปภาพ Avatar)
- **ระบบสั่งซื้อ:** เลือกสินค้าลงตะกร้า, ระบบ Checkout, อัปโหลดสลิปยืนยันการชำระเงิน
- **ระบบหลังบ้าน (Admin Dashboard):** - จัดการข้อมูลสินค้า (เพิ่ม/ลบ/แก้ไข)
  - จัดการสต็อกสินค้า (ขนาด, สี, จำนวน)
  - ตรวจสอบรายการสั่งซื้อและสลิปการโอนเงิน
  - จัดการข้อมูลผู้ใช้งาน

## 🛠 เทคโนโลยีที่ใช้ (Tech Stack)
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5
- **Backend:** PHP (Native)
- **Database:** MySQL
- **Tools:** XAMPP, Git Bash

## 🚀 วิธีติดตั้งและใช้งาน (Installation)
1. Clone โปรเจคนี้ลงเครื่อง
2. นำโฟลเดอร์ `geb_sneakers` ไปวางใน `htdocs` ของ XAMPP
3. สร้าง Database ใน phpMyAdmin ชื่อ `geb_sneakers`
4. Import ไฟล์ `geb_sneakers.sql` เข้าไปในฐานข้อมูล
5. แก้ไขไฟล์การเชื่อมต่อฐานข้อมูลที่ `inc/connect.php`
6. เปิด Browser แล้วพิมพ์ `localhost/geb_sneakers`
